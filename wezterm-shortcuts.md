# WezTerm Keyboard Shortcuts

**Leader key:** `Ctrl+a` (1 second timeout)

## Tab Management

| Shortcut | Action |
|----------|--------|
| `Ctrl+Shift+t` | New tab |
| `Ctrl+Shift+w` | Close tab (with confirm) |
| `Ctrl+Tab` | Next tab |
| `Ctrl+Shift+Tab` | Previous tab |
| `Alt+1` - `Alt+9` | Switch to tab 1-9 |

## Pane Management

### Splitting

| Shortcut | Action |
|----------|--------|
| `Ctrl+Alt+\` | Split horizontal |
| `Ctrl+Alt+-` | Split vertical |

### Navigation

| Shortcut | Action |
|----------|--------|
| `Ctrl+Shift+Arrow` | Navigate panes (arrow keys) |
| `Leader` then `h/j/k/l` | Navigate panes (vim style) |

### Resizing

| Shortcut | Action |
|----------|--------|
| `Leader` then `Shift+H` | Resize left |
| `Leader` then `Shift+L` | Resize right |
| `Leader` then `Shift+K` | Resize up |
| `Leader` then `Shift+J` | Resize down |

### Other Pane Actions

| Shortcut | Action |
|----------|--------|
| `Alt+w` | Close pane (with confirm) |
| `Leader` then `z` | Toggle pane zoom |

## Font Size

| Shortcut | Action |
|----------|--------|
| `Ctrl+Shift++` | Increase font size |
| `Ctrl+-` | Decrease font size |
| `Ctrl+0` | Reset font size |

## Copy/Paste

| Shortcut | Action |
|----------|--------|
| `Ctrl+Shift+c` | Copy to clipboard |
| `Ctrl+Shift+v` | Paste from clipboard |
| `Right click` | Paste from clipboard |

## Scrolling

| Shortcut | Action |
|----------|--------|
| `Shift+PageUp` | Scroll up one page |
| `Shift+PageDown` | Scroll down one page |

## Misc

| Shortcut | Action |
|----------|--------|
| `Ctrl+Shift+,` | Open config in nvim |
| `Leader` then `r` | Reload configuration |
| `Ctrl+Shift+p` | Command palette |
| `Ctrl+Shift+f` | Search mode |
| `Leader` then `[` | Copy mode (vim selection) |
| `Ctrl+click` | Open link |
